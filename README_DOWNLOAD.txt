DOWNLOAD INSTRUCTIONS
=====================
If the direct ZIP download links are not working, download the part files below.
Each file is a base64-encoded chunk of the ZIP. After downloading all parts, use
recombine.py to rebuild the original ZIP.

For production website ZIP:
- kadslabs_website_part1.txt
- kadslabs_website_part2.txt
- kadslabs_website_part3.txt

For full source ZIP:
- kadslabs_source_part1.txt
- kadslabs_source_part2.txt

Steps:
1. Download all part files for the ZIP you want.
2. Also download recombine.py.
3. Put them in the same folder.
4. Run: python3 recombine.py

Or use an online base64-to-file decoder for each part and concatenate the raw bytes.
